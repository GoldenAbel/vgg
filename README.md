# VGG

## Directory Structure

- `./`: project root
  - `/script`: useful scripts
  - `/src`: source code
  - `/data`: training and testing dataset
    - `/train`
    - `/test`
  - `/label`: label for training
  - `/dest`: output (if any)
    - `/log`: log for tensorboard
    - `/param`: store check points
  - `/exchange`: exchange data between server and client
