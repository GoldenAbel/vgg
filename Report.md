# Reading Report
## 研究目的 

探究随着层数的增加，卷积网络的效果的发生什么变化

## 网络架构 

### 图像预处理
- 输入图像大小为224×224

### 卷积层

- 卷积步长为1
- 3×3的过滤器，其中有一个配置使用了1×1的过滤器

### Max-pooling层
- 5层
- 2×2窗口，步长为2

### 全连接层

- 前2层每个有4096通道
- 最后1层有1000通道

All hidden layers are equiped with rectification non-linearity.

### 配置（Network D 16 weight layers）
- 13 卷积层 + 3 全连接层
- 卷积层宽度包括64, 128, 256, 512
- 参数数量：138 million
- 3×3 filter width

## 分类框架

- 图片大小：

- Network weight初始化：
